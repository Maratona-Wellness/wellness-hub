# Project Theme
This document outlines the theme specifications for the project, including colors, typography, spacing, border radius, and shadows.

**Project Name:** Maratona Qualidade de Vida Wellness Hub
*Aliases:* MQV Wellness Hub, Maratona Wellness Hub, Maratona QV Wellness Hub, etc.


## Colors

70%: This is mostly used on primary elements, such as background. "70%" of the screen color.
20%: This is used for secondary elements, such as cards or secondary buttons. "20%" of the screen color.
10%: This is used for accents and highlights, such as links or active states. "10%" of the screen color.

70% Color: #efefef;
20% Color: #444444;
10% Color: #97181B;

## Typography

Font Family: Varela Round, 'Helvetica Neue', Helvetica, Arial, sans-serif;
Base Font Size: 16px;
Line Height: 1.5;
Headings Font Weight: 700;
Body Font Weight: 400;
Bold Font Weight: 600;

## Spacing

Base Spacing Unit: 8px;
Small Spacing: 8px;
Medium Spacing: 16px;
Large Spacing: 32px;
Extra Large Spacing: 64px;

## Border Radius

Small Border Radius: 4px;
Medium Border Radius: 8px;
Large Border Radius: 16px;
Extra Large Border Radius: 32px;

## Shadows

Medium Shadow: 0 3px 6px rgba(0, 0, 0, 0.16), 0 3px 6px rgba(0, 0, 0, 0.23);
Large Shadow: 0 10px 20px rgba(0, 0, 0, 0.19), 0 6px 6px rgba(0, 0, 0, 0.23);
Extra Large Shadow: 0 15px 25px rgba(0, 0, 0, 0.22), 0 10px 10px rgba(0, 0, 0, 0.24);
